# MAYA Pledge Manager - Changelog

All notable changes to the MAYA Pledge Manager will be documented in this file.

---

## [1.4.0] - 2026-02-03

### Added
- Full support for late backers (backed after campaign ended)
- Late backers now see retail pricing for additional purchases
- Comprehensive data sync for all late backer records

### Fixed
- Dashboard now correctly displays pledge tier and items for late backers
- Upgrade options now show based on tier hierarchy (not just price)

---

## [1.3.0] - 2026-02-01

### Added
- Editable shipping address from dashboard (country remains locked)
- Order confirmation view after checkout completion
- ASIA shipping zone with rates for 44 countries

### Changed
- Improved dashboard layout for completed orders
- Better display of Kickstarter items and add-ons

---

## [1.2.0] - 2026-01-29

### Added
- Case-insensitive email login (no more case sensitivity issues)
- Google Sheets integration for automatic order logging

### Fixed
- "Send a Code Instead" button now works correctly
- Cart sync issues when navigating between pages

---

## [1.1.0] - 2026-01-27

### Added
- Support for dropped backers to complete failed Kickstarter payments
- Immediate payment option for Indian backers
- SetupIntent flow for secure card saving (non-Indian backers)

### Fixed
- Pledge images not loading on mobile
- PIN entry spacing on mobile devices

---

## [1.0.0] - 2026-01-25

### Added
- Initial release of MAYA Pledge Manager
- Kickstarter backer authentication via PIN/OTP
- Pledge upgrades with backer pricing
- Add-on purchases
- Shipping cost calculator by region
- Stripe payment integration
- Order confirmation emails

---
